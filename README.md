# Board Games Project

## Overview

This project is a C++ application that includes four different board games:

* 4x4 Tic-Tac-Toe
* Pyramid Tic-Tac-Toe
* Numerical Tic-Tac-Toe
* Obstacles Tic-Tac-Toe

The project was built using Object-Oriented Programming (OOP) concepts such as classes, inheritance, polymorphism, and encapsulation.

It supports both human players and computer players, with some games including AI-based gameplay using Minimax and board evaluation techniques.

---

## Features

* Multiple board games in one application
* Human vs Human mode
* Human vs Computer mode
* AI player for selected games
* Input validation and error handling
* Organized and reusable OOP structure
* Console-based user interface

## Games Included

### 1. 4x4 Tic-Tac-Toe

A larger version of the traditional Tic-Tac-Toe game played on a 4x4 board.

### 2. Pyramid Tic-Tac-Toe

A Tic-Tac-Toe variation played on a pyramid-shaped board with different winning possibilities.

### 3. Numerical Tic-Tac-Toe

A number-based Tic-Tac-Toe game where players use numbers instead of symbols.

### 4. Obstacles Tic-Tac-Toe

A Tic-Tac-Toe variation that includes blocked cells and obstacles on the board.

---

## How to Run

1. Open the project in your C++ IDE.
2. Compile and run the project.
3. Choose one of the available games from the menu.
4. Select the game mode and start playing.
